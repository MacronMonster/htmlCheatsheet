What is HTML?

HTML stands for HyperText Markup Language. It is used to give a website content and structure. This cheatsheet will serve as a source of HTML elements when you need to program in this language.

Media

Tag Name Purpose
<img src = "image_name.png"> Image To embed images in a page.
Attributes: The src attribute is mandatory since it indicates the image URL. Note: Do NOT put a closing tag on a line break element!
<video src = "video_name.mp4" controls > Error Message </video> Video To embed videos in a page.
Attributes: The src attribute is mandatory since it indicates the image URL. controls attribute will display video controls in media player. All content between opening and closing tag will be shown in case video does not play
<img src = "image_name.png" alt = "Image not supported" /> alt Attribute The alt attribute indicates what the browser will display when the image cannot be uploaded
<a href = "websitelink.com"> Visit this site!</a>
or
<a href = "websitelink.com"> <img src = "image_name.png">Click this image! </a>		 Anchor Element Used to create hyperlinks in an HTML document.
Attributes: href attribute specifies where the link will take the user (websites, files, other places on same page, etc). All content between opening and closing tags is what the hyper link is applied to
<a href = "websitelink.com" target = "_blank"> Visit this site in a new tab!</a> target Atrribute The target attribute specifies where the target should be opened. If it equals "_blank", it will be opened in a new tab or window
<audio src = "audio_name.mp3"></audio> Audio Implements audio into website
<embed src = "file.gif"/> Embed Implements any type of media into website
<figure> <img src = "image_name.png"> <figcaption>Caption of image here</figcaption> </figure> FigCaption Element Used to describe media incapsulated within figure element. Usually placed within figure element to group the media and it's description
<link href = "file.css" rel = "stylesheet" type = "text/css"> Links Element Used to link HTML file to external sources link CSS files
Attributes: href attribute specifies URL to external source. rel attribute specifies relationship of linked document to current document. type attribute defines type of conent being linked

Structural Elements

Tag Name Purpose
<!DOCTYPE html> Document Type Declaration Important! Tells the browser what type of ducument to expect and the version of HTML being used
<head> ... </head> Head Element Contains general information about the site that will not be displayed in the on the page
ex: links to style sheets or the page name
<title> ... </title> Title Element Contains text that defines the title of an HTML document found in title bar or tab. Belongs in head element
Whitespace The rows between code inputted to make it easier to read; will not be read by browser
<html> ... </html> HTML Element The root of an HTML document. Should be found after DOCTYPE declaration.
<-- Comments --> Comments Used to decribe a part of the code or give extra details on project. Can span one or multiple lines
<element_tag id = "id Name"> ... <element_tag> Unique ID Attribute An ID that is not read by browser and can be called upon by JavaScript and CSS to manipulate, format, and preform specific instructions to only that element
<element_tag class = "id Name"> ... <element_tag> Class Attribute The class attribute is used to specify a class for a specific HTML element. When called by JavaScript and CSS programs to manipulate and format the code
<body> ... </body> Body Element Represents content of HTML document that will be rendered by browser
<div> ... </div> Div Element Used as a container that divides an HTML document into sections containing flow content like headings, links, images, etc. Short for "division"
<section> ... </section> Section Element Used to describe HTML elements that share the same theme
<article> ... </article> Article Element Used to decribe HTML elements that make sense on their own like blogs and articles. Usually placed within a section element when content follows the same theme but can be independent. Does not have to be placed within a section element
<aside> ... </aside> Aside Element Used to mark additional information that enhances main content; not necessary information
<figure> <img src = "image_name.png"> </figure> Figure Used to incapsulate media such as an image, diagram, or code snippet

Tables

Tag Name Purpose
<td> ... </td> Table Data Adds a cell of data to a table. Must be nested within <tr> element
<tr> ... </tr> Table Row Adds rows of data to a table before adding data and headings
<thead> ... </thead> Table Head Defines table headings of table columns within table rows
<th rowspan = "#<65534"> ... </th>
or
<td rowspan = "#<65534"> ... </td>		 rowspan Attribute Indicates number of rows that particular cell should span in the table. Automatically set to 1 and can go up to 65534
<tbody> ... </tbody> Table Body Contains all table data other than the table heading and table footer
<th colspan = "#<1000"> ... </th>
or
<td colspan = "#<1000"> ... </td>		 colspan Attribute Indicates number of columns that particular cell should span in the table. Automatically set to 1 and can go up to 1000
<tfoot> ... </tfoot> Table Footer Uses table rows to give footer content or summarize content at the end of the table
<table> ... </table> Table Element Tells browser that a table will be rendered. Used to reprensent a two dimensional table made of rows and columns
<th> ... </th> Table Heading Element Adds titles to the rows and columns of a table. Must be nested within <tr> element
